Problem Statement:

Given an Array of elements, display the frequency of each of those elements in decending order

Input Format:
Line1: Number of elements in the array n
Line2: n space separated integers

Output Format:
k lines where each line contains the element and frequency separated by a whitespace

Note: Output to be displayed in decending order of frequency. If two elements have same frequency, the element which comes 
first in the array has to be displayed first

Constraints: 
0 <= n <= 10^5


sample input:
9
12 23 34 12 23 23 34 15 18

sample output:
23 3
12 2
34 2
15 1
18 1
